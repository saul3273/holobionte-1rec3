## 🔗 Checklist de Coordinación y Compatibilidad

### Pre-Integración
- [ ] Se ha revisado la [estructura del proyecto](../ESTRUCTURA_LISTA.md)
- [ ] Se consultó [ETIMOLOGIAS.md](./ETIMOLOGIAS.md) para nomenclatura consistente
- [ ] Se verificó compatibilidad con [INSTRUCCIONES_UNIVERSALES.md](../INSTRUCCIONES_UNIVERSALES.md)
- [ ] Se revisaron [capacidades estratégicas](./capacidades-estrategicas.md) del holobionte
- [ ] Se identificaron posibles conflictos con simbiontes existentes

### Documentación
- [ ] Archivo de simbionte creado en `/memoria/simbiontes/[NOMBRE].md`
- [ ] Actualizado el registro en `/memoria/simbiontes/README.md`
- [ ] Documentadas las instrucciones específicas del simbionte
- [ ] Creados ejemplos de uso y casos de aplicación
- [ ] Añadidas referencias cruzadas con otros documentos relevantes

### Integración Técnica
- [ ] Configuración de acceso probada y funcional
- [ ] Variables de entorno configuradas
- [ ] Dependencias instaladas y verificadas
- [ ] Pruebas de conectividad completadas exitosamente
- [ ] Sistema de fallback configurado (si aplica)

### Coordinación con el Holobionte
- [ ] Protocolos de comunicación establecidos
- [ ] Mecanismos de handoff entre simbiontes definidos
- [ ] Sistema de priorización configurado
- [ ] Estrategia de resolución de conflictos definida
- [ ] Monitoreo y logging implementados

### Gobernanza y Ética
- [ ] Revisión de implicaciones éticas completada
- [ ] Políticas de privacidad y datos verificadas
- [ ] Mecanismos de auditoría en su lugar
- [ ] Plan de desactivación/sustitución definido
- [ ] Documentación de decisiones en [DECISIONES.md](../DECISIONES.md)

### Post-Integración
- [ ] Periodo de prueba de 7 días completado
- [ ] Feedback inicial documentado
- [ ] Ajustes de configuración realizados
- [ ] Actualizado [ESTADO_ACTUAL.md](../ESTADO_ACTUAL.md)
- [ ] Comunicado al colectivo en asamblea o canal
